# JSON Refiner – Advanced Edition

## 📌 Project Overview
JSON Refiner is a Python-based tool developed in Google Colab that processes, validates, and formats raw JSON data into structured and clean output.

## 🚀 Key Features
- JSON validation
- Error handling for invalid JSON
- Pretty formatting (indentation)
- Data cleaning and filtering
- Export refined JSON file

## 🛠 Technologies Used
- Python
- Google Colab
- JSON Library

## 📂 Project Files
- JSON_Refiner_Advanced.ipynb
- sample_input.json
- refined_output.json
